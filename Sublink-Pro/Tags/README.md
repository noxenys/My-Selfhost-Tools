# 🏷️ Sublink Pro - 标签管理模块

本模块用于快速初始化 Sublink Pro 的标签系统，支持全量导入、核心导入及一键清理。

## ⚠️ 核心维护笔记 (必读)

1. **入组关键**：添加标签接口必须使用 `groupName` 字段，否则标签无法在 UI 中正确显示蓝色分组标识。
2. **清理逻辑**：删除接口仅支持 `DELETE` 方法，且标签名需通过 URL 参数 `?name=` 传递。
3. **安全规范**：由于本仓库为 **Public**，脚本中使用了 `input()` 动态获取 Token，**严禁将真实的 Token 直接写入代码上传**。

## 🚀 脚本列表说明

| 文件名 | 功能说明 |
| :--- | :--- |
| `clean_tags.py` | 一键清空所有标签 (删除 46 个常用预设) |
| `sync_core_20.py` | 导入 20 个精选核心标签 (速度/延迟/地区/流媒体/稳定性) |
| `sync_full_46.py` | 导入 15 组共 46 个全量细分标签 |

## 🛠️ 使用步骤
1. 打开浏览器登录 SLP 后台，按 **F12** 进入网络面板。
2. 随便操作一个标签，找到请求头中的 `Authorization: Bearer <TOKEN>`。
3. 运行脚本，在提示符处粘贴 Token 即可。
